自己紹介

技術そのものだけでなく、「どんな課題を解決したいのか」「どの手段が最適か」といった企画・問題定義の部分から考えることを大切にしています。
個人開発では、「なぜこのアプリを作るのか」「誰の役に立てるのか」を明確にしながら設計や実装を行っています。

また、リーダーや発表の機会を多く経験したことで、使う技術の選定理由やシステム構成について、自分の言葉で伝える力も意識して磨いてきました。
単に動くものを作るだけでなく、**なぜそのように作るのか**を説明できるエンジニアを目指しています。
