研究

スキーマ駆動開発 — OpenAPIとgRPCの比較研究

2025年4月14日 〜 2025年7月30日 3年前期
  • OpenAPI / gRPC / コードファーストの3手法を比較
  • 使いやすさ、型安全性、拡張性、学習コスト、チーム開発との相性などの観点から評価
  • 各手法で読書記録アプリのAPIを実装し、開発プロセスや生成コードを比較
  • OpenAPIにはSwagger Codegen、gRPCにはProtocol Buffersを用いてスキーマ定義
  • 結果をレポート・スライドにまとめ、スキーマ駆動開発の有用性を検討

単に便利な技術というだけでなく、「どのような場面で有効か」「導入時にどんな工夫が必要か」といった観点から考察しました。 また、ドキュメント設計と実装を分離することでチーム内の認識齟齬を減らすといったスキーマ駆動開発のメリットを、実体験として得ることができました。
